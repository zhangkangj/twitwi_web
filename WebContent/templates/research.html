For Researchers

Open data?

As data scientists, we get great intuitions from simple things like counting numbers and basic arithmetics. Yet, in the days of Big Data, these seemingly simple tasks aren't simple after all. For example, counting Twitter mentions requires a huge amount of work to build a robust database system and an analytic engine. We have gone through that pain, and we want to make this easy for everyone else. Therefore, we'd like to release our data for academic research to the extent permitted by Twitter Terms of Service.

What is available?

Currently, number of mentions of Obama and Romney in each state / about each topic on each day since 2012. That's what you see on the website. Please read about for more details.

How reliable?

There are many Twitter research groups and each of them collect and analyze data in its own way. The difficulty of data verification has casted suspicion on Twitter research. Thus, we'd to provide an open, reliable and unbiased data source. For any statistics we present, we can back it up with the exact tweets and algorithms used. Upon request, we will release the tweet IDs for independent verification.

What can I use it for?

Academic research or other not-for-profit purposes. For other usage, please let us know. We have to ensure that your usage complies with Twitter's Terms of Service so that the data are always openly available.

How to get it?

We are working on an open API end point. But if you want to play with the data now, please contact us.

We do have the capacity to track a lot more things. Tell us how great your idea is.
